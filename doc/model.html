Quantum Wave Interference model atlas

Three laboratories, four sources, one wave grammar.

A redesigned reference for the Quantum Wave Interference model. It tracks the Experiment, High Intensity, and Single Particles screens as related instruments: each scene keeps its own source, slits, detector data, and snapshots while sharing the same core story of wavelength, measurement, coherence, and hits.

Overview

The simulation is organized as source-type scenes. A scene is not merely a selected label: it owns source settings, slit settings, detector-screen data, and up to four snapshots. Changing from photons to electrons switches to a different preserved scene.

Experiment

Detector-pattern experiment with overhead apparatus, front-facing slits, screen distance, intensity, and detector-screen zoom.

High Intensity

Continuous analytical waves through none or double slit barriers, accumulating average intensity or sampled hits.

Single Particles

One Gaussian packet at a time, with packet timing, auto-repeat, detector tool probability, and localized hits.

Common physics hinge: photons use the wavelength slider directly. Matter particles derive wavelength from the de Broglie relation.

lambda = h / ( m * v ), h = 6.626e-34 J*s

Solvers and Detection

The model intentionally separates visual scale, physical quantities, and detector sampling.

1. SourcePhoton wavelength or particle speed determines effective wavelength.
2. ScaleWave region is sized for about 15 default wavelengths on High Intensity and Single Particles.
3. SlitsPhysical slit values are mapped into display geometry for the analytical solvers.
4. FieldAnalytical solvers evaluate continuous waves or Gaussian wave packets.
5. DetectionIntensity or probability distributions produce detector-screen patterns and hits.
Experiment detector pattern

Uses a Fraunhofer model. Both slits open multiply a double-slit cos^2 interference factor by a single-slit sinc^2 diffraction envelope. One covered slit leaves the single-slit diffraction envelope. Which-path detectors remove coherence and eliminate double-slit fringes.

High Intensity detection

Average Intensity accumulates after the wavefront reaches the detector. Hits mode samples from the detector probability distribution at 40 detector-screen hits per model second. Slit detectors create decoherence events at 5 conceptual particles per model second.

Single Particles detection

Detector-screen and slit-detector timing are sampled from the same packet-weight timing curve. If a slit detector clicks, it re-emits a downstream packet from the selected slit. The detector tool integrates probability inside a circular region and can either detect or remove that probability from the packet.

Accessible Reading

The model is shaped to support both visual rendering and nonvisual explanation. Dynamic descriptions can read the same facts a teacher would point to visually: source type, slit setting, detector mode, hit stage, and pattern kind.

Stable state

Screen summaries and current details can describe what exists now: selected source, setup, detector pattern, tools, and hit count.

Responsive state

Context responses can describe what changed: source started, slit separation changed, packet detected, or detector screen cleared.

Useful invariant: if visual state changes because of a source, slit, or detection action, the accessible model should expose the same conceptual change: pattern kind, detector progress, hit count, or packet state.
